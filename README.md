# Module1_Calculator
Module1 for CSU Class MS in AI/ML
Part 1:
Ask the user to input two numbers (num1 and num2). Given those two numbers, add them together to find the output. Also, subtract the two numbers to find the output.
Part 2:
Ask the user to input two numbers (num3 and num4). Given those two numbers, multiply them together to find the output. Also, divide num1/num2 to find the output.
Checks for divisions by zero.
